Welcome to IGOS Nusantara SDK Pages.

IGN SDK is a software development kit designed specifically for application developers in IGOS Nusantara. But now also available for other Linux distros such as Arch Linux and Debian

Wiki

  • https://github.com/anak10thn/ignsdk-qt/wiki

Build

$ sudo yum install qt-devel qtwebkit-devel qjson
$ cd ign-sdk-directory
$ qmake
$ make
